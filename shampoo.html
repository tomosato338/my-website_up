シャンプー「Septem」Webサイト

サイトはこちら

概要

カラーやパーマによるダメージ、ロングヘア特有のパサつきやまとまりに悩む20代女性のための、架空のシャンプーブランド「Septem（セプテム）」を紹介する公式サイトです。髪本来の美しさを取り戻し、指通りの良い艶やかなスタイルを叶えることをコンセプトに、商品の特長やこだわり成分、キャンペーンなどを掲載しています。

サイトの目的

①新規顧客獲得 ②購入ボタンのクリック　③Twitterキャンペーンボタンのクリック　④商品の認知度向上

ターゲット

性別 女性
年齢 20代
悩み カラーで傷んだダメージ毛を修復したい。
ロングヘア―にしてからパサつき・まとまりが気になり始めた。
ペルソナについて▼

サイト構成について

パララックススクロールを採用することで、洗練されたおしゃれな印象を演出しています。PC版では、左サイドに目次、右サイドに購入ボタンを配置し、ユーザーがスクロールしながらでもスムーズに購入できるように設計しました。PC・タブレット・スマートフォンの各デバイスに対応したレスポンシブデザインを実装しています。

デザインについて

髪にやさしいシャンプーであることを伝えるため、温かみのある配色を採用し、写真ではなくイラストを用いてデザインしました。フォントには、洗練された上品さを演出する「游明朝」を使用し、キャンペーンバナーには視認性と強調性を高めるため「游ゴシック」を組み合わせています。商品の特長が一目で伝わるよう、jQueryで水泡が揺れるアニメーションを加え、最後まで飽きさせることなく購入ボタンへ誘導し、クリック数の向上を図りました。また、ペルソナが重視していた「ボトルデザイン」や「髪質への適合性」に応えるため、成分情報が直感的に伝わるレイアウト・構成を意識してデザインしています。

使用ツール

XD / Photoshop / Illustrator / HTML / CSS / jQuery
